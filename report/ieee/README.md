# IEEE Report Files

This folder contains the Overleaf / LaTeX version of the final report.

## Files

- `main.tex`: main LaTeX report file
- `references.bib`: BibTeX references
- `figures/`: place report figures here

## How to use Overleaf

1. Create a new Overleaf project.
2. Upload `main.tex` and `references.bib`.
3. Upload the images from `report/figures/` into a folder named `figures`.
4. Set the main document to `main.tex`.
5. Compile.

If a figure is missing, either upload the required PNG file or comment out the corresponding `figure` block in `main.tex`.
